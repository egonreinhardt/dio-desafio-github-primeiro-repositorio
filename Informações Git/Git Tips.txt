Download Git: https://git-scm.com/downloads

-Ao clicar com o botão direito em uma pasta e escolher "Git Bash here", o Git Bash vai abrir com a pasta em questão já selecionada;

-Comando "git clone [link do repopsitório]" clona o repositório na pasta selecionada;

-Comando "git status" mostra se há algum arquivo na pasta pendente para ser commitado;

-Comando "git add ." adiciona tudo o que está untracked (pendente) ao status staged (está no repositório local);

-Comando 'git commit -m "comentário"' adiciona arquivos de stage para commited (gera o commit localmente);

-Comando "git push origin main" publicará o commit local.
